## 📖 项目简介

Open TCM 是一个开源的中医药知识库系统，旨在为中医药文化的传播和学习提供现代化的数字平台。本系统收录了大量中医方剂、中药材等信息，并提供便捷的搜索和查询功能。

## ✨ 主要特性

- 📚 丰富的中医方剂数据库
- 🌿 详细的中药材信息
- 🔍 强大的全文搜索功能
- 💻 现代化的用户界面
- 📱 响应式设计，支持多端访问

## 🛠️ 技术栈

- 前端：Next.js, React, Tailwind CSS
- 后端：Payload CMS
- 数据库：PostgreSQL
- 搜索引擎：内置搜索插件
- 部署：Vercel
